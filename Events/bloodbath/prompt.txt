Last player standing wins in Bloodbath arena.

Arena shrinks over time. Spawn pressure increases.

Players start with full T09 gear and all abilities unlocked.

PvP is enabled. Use pvp.enable action.

Use spawn.wave to add enemy waves. Use shrink.zone to shrink playable area.

Use announce and notification for player messaging.

Use healing actions to restore player health between rounds.

Use the blood.change action to modify player blood type if needed.
